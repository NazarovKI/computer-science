Программа по информатике которую пытался опробовать в школе №71, 
это Календарно-тематическое планирование пока проект. 
Но проект это всегда исследование, 
поэтому требует ежедневной работы.
Блог показывает по дням - 
что для исследования сделано, 
В чем приращение, какими новыми знаниями мы овладели.
Каждый урок - предмет коллекции, 
который имеет характеристики записанные в yaml, 
Они обычные в системе ФГОС и "школа России".
48 уроков цель конечная цель которых - развитие алгоритмического мышления у детей. 
Правильно ли сформулирована цель?
Внедрение указанной программы не исключает подготовку к ЕГЭ, напротив - предлагает освоение следующих инструментов:

1. markdown язык разметки который позволит без усилий превращать полученную на любых уроках информацию в знания работая за компьютером;
2. LaTeX и tikz для Повторения полученные на уроках математики и химии формул, и  углубения их понимания учащимися;
3. Диаграммы mermaid лёгкий способ создания схем, графической записи алгоритмов для уроков истории, информатики, экономики, при ответе можно использовать электронную доску и проектор;
4. Гост .... Умение работать с источником информации, указание библиографической ссылки позволит вернуться к источнику в чтобы лучше обдумать сказанное и избежать плагиата при защите проекта;
5. reveal.js или marp для  презентаций из конспектов markdown;
6. Obsidian или code oss для объединия всёго в единую рабочую среду, в индивидуальную базу знаний в парадигме personal/index.
7. Github для совместной работы над проектами где каждая строка - вклад. И каждая строка кода имеет историю, буквально как в библии!
